# Как проверить работу AI-инсайтов

## 1. Убедиться, что ключ подключён

- В Vercel: **Project → Settings → Environment Variables** должна быть переменная **`OPENAI_API_KEY`** (для Production и при необходимости Preview).
- После добавления/изменения переменной сделайте **Redeploy** проекта.

## 2. Открыть дашборд

- Откройте задеплоенный сайт на Vercel (например `https://ваш-проект.vercel.app`).
- Дождитесь загрузки данных (должен появиться статус «GA4 API» или данные из демо).

## 3. Перейти в раздел с инсайтами

- Откройте вкладку **Content** (Контент).
- Прокрутите вниз до блока **«Выявленные закономерности и инсайты»** и **«Стратегические рекомендации»**.

## 4. Что вы должны увидеть, если ИИ работает

- На 1–3 секунды появится текст: **«Анализ текущих данных с помощью ИИ…»** (или «Analyzing data with AI…»).
- Затем подставятся:
  - **Краткий анализ** по вашим реальным данным (пики, каналы, волатильность и т.п.).
  - **Нумерованный список рекомендаций** (3–5 пунктов), завязанных на этих данных.
- Справа от заголовка «Выявленные закономерности и инсайты» появится **бейдж «ИИ»** (или «AI» на английском) — это значит, что текст сгенерирован моделью.

Если бейдж **«ИИ» не появляется**, а текст похож на общий шаблон — значит запрос к API не прошёл, и показывается встроенный (не ИИ) анализ.

## 5. Проверка через DevTools (по желанию)

1. Откройте **Developer Tools** (F12 или правый клик → «Просмотреть код»).
2. Вкладка **Network** (Сеть).
3. Обновите страницу или нажмите **«Обновить»** на дашборде.
4. В фильтре введите **`insights`**.
5. Найдите запрос **`insights`** методом **POST**:
   - **Status 200** — запрос к ИИ прошёл, в ответе должны быть поля `summary` и `recommendations`.
   - **Status 503** — ключ не задан или ошибка на стороне API (проверьте `OPENAI_API_KEY` и логи в Vercel).

## 6. Локальный запуск (если нужно проверить до деплоя)

```bash
# Установить Vercel CLI (если ещё не установлен)
npm i -g vercel

# В корне проекта запустить режим разработки (подхватит serverless-функции и переменные из Vercel)
vercel dev
```

В браузере откройте указанный локальный адрес (например `http://localhost:3000`), перейдите на вкладку Content и проверьте появление бейджа «ИИ» и текста инсайтов.

---

**Итог:** если после загрузки данных на вкладке Content вы видите бейдж **«ИИ»** и уникальный анализ под ваши цифры — AI-инсайты работают.
